# 🚀 SmartEmployee - Employee Management System

SmartEmployee is a modern and professional Employee Management System built using Laravel, Blade, Tailwind CSS, and MySQL.

The platform helps organizations efficiently manage employees, departments, attendance workflows, and administrative operations through a clean and responsive dashboard.

---

# ✨ Features

- 🔐 Secure Authentication System
- 👨‍💼 Employee Management (CRUD)
- 📊 Dynamic Admin Dashboard
- 🏢 Department Management
- 📈 Real-Time Employee Statistics
- 📱 Fully Responsive Design
- ⚡ Fast Performance with Vite
- 🎨 Professional Tailwind CSS UI
- 🔎 Search & Pagination Ready
- 🛡️ Role-Based Structure Ready
- 📋 Employee Status Tracking

# ⚙️ Installation Guide

## Clone Repository

```bash
git clone https://github.com/Princehoon16/helloiocian.git
```

## Move into Project Directory

```bash
cd helloiocian
```

## Install Dependencies

```bash
composer install
npm install
```

## Create Environment File

```bash
cp .env.example .env
```

## Generate Application Key

```bash
php artisan key:generate
```

## Configure Database

Update `.env` file:

```env
DB_DATABASE=smartemployee
DB_USERNAME=root
DB_PASSWORD=
```

## Run Migrations

```bash
php artisan migrate
```

## Start Development Server

```bash
php artisan serve
npm run dev
```
